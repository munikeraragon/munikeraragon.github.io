Home Security System
This System provides Infrared Motion Detection as well as live video transmission to a website. The hardaware components utilized in this project consist of a raspberry pi model 3, lux sensor TSL2561, and a raspberry pi Infrared Night Vision camera. These components interact with created C scripts which analyze the different luminosity measurements to determine the proximity of objects. This System also implements various video encoding frameworks such as FFmpeg and jsmpeg.
